sudoku-solver
=============

A simple non-backtracking sudoku puzzle solver implementing basic human
strategies.


Input files are in the format:

    $ cat data/guardian.sud 
       93    
     72  49  
    3      1 
      42     
    18     69
         13  
     4      1
      16  79 
        58   

Running
=======

    $ ant compile jar
    $ ant run
